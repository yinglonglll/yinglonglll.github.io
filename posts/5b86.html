1.1 链接跳转

  • 百度：链接调用方法

  • 锚点：

    1. 原生方法：仅当目标为标题时才剩下，若处于X级标题也会失效，属于自身拥有跳转属性的方法，同时字体增大，影响美观，不建议使用

      // 代码：实际使用操作 Ctrl + 左键点击
# titleA
- [toA](#titleA)
      • toA

    2. html方法：建议使用，属于附属属性，不影响原先字体大小

      // 在目标处进行a标签的标记，
# <a id ="titleB">titleB</a>或<a name ="titleB">titleB</a>
- [toB](#titleB)
      • toB

  • 图片引用：建议使用图床，避免图片资源的丢失，也可以在偏好设置–>图形 里面设置上传服务

    1. typora原生方法：进行图片缩放后，可能存在无法左对齐的情况

    2. html方法：可左对齐图片、限制图片大小；但是无法全局替换关键字
